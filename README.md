# PI - Faculdade das Americas

* code by gaqno 
* expo project based
* testhttp extension for test requests.
* front end based on react native library
* back end based on prisma as ORM, MySQL as DB, Next.JS as backend framework
* ux by julia sousa

# How to run the project?

* front end 
-- bash
cd /front
yarn
yarn start

* back end 
-- bash
cd /back
yarn
yarn start

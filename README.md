## Installation

Please use yarn to install

```bash
$ yarn install
```

## Running the app

For a development environment, you can use docker-compose

```bash
# docker-compose up
```

or run directly. you will need to install mysql and add it to your /etc/hosts on 'database' alias

```bash
# development
$ yarn start
```
